# URL

A class that makes up a url

(web://wow.bouncy.flf/subdir/subdir2/shocker.txt?test=wow\&crazy=noway)

```json5
{
    "scheme": "web",
    "domain_name": "bouncy",
    "domain_top": "flf",
    "domain_sub": "wow",
    "path": "subdir/subdir2",
    "params": {
        "test": "wow",
        "crazy": "noway"
    },
    "file_name": "shocker.txt"
}
```
