# /search\_users

## About

Returns an array of users that have your query in their username

## Parameters

| Parameter | Description                                                       |
| --------- | ----------------------------------------------------------------- |
| limit     | Optional. The number of users to return, default is 20, max is 50 |
| q         | Your text query                                                   |
