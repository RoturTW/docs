# /search\_posts

## About

Returns the an array of posts that have your query in their content

## Parameters

| Parameter | Description                                                       |
| --------- | ----------------------------------------------------------------- |
| limit     | Optional. The number of posts to return, default is 20, max is 50 |
| q         | Your text query                                                   |
