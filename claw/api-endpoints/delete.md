# /delete

### About

Deletes a post on Claw if the authenticated user is the creator.

### Parameters

#### auth

{% embed url="https://claw.rotur.dev/delete?auth=YOUR_AUTH_KEY&id=POST_ID" %}

A required user authentication key.

#### id

{% embed url="https://claw.rotur.dev/delete?auth=YOUR_AUTH_KEY&id=POST_ID" %}

The ID of the post to delete.
